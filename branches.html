Tableaux (Arrays) - JavaScript

Un **tableau** est une structure de données qui stocke une collection ordonnée de valeurs. Ces valeurs peuvent être de n'importe quel type (nombres, chaînes de caractères, objets, ou même d'autres tableaux). Les éléments sont accédés par leur **index numérique**, qui commence toujours à **0**.

Création et Accès
let arr = []; // let arr = new Array();
Déclaration (Littérale) 
let fruits = ["Pomme", "Poire", "Banane"];

C'est la méthode la plus courante et recommandée pour créer un tableau.

Accéder à un élément 
let premier = fruits[0]; // "Pomme"

Accès via l'**index** (position). Le premier élément est toujours à l'index **0**.

Taille du tableau 
let taille = fruits.length; // 3

La propriété **.length** retourne le nombre d'éléments dans le tableau.

Modification du Tableau (Méthodes Mutables)
.push(), .pop(), .splice()
.push(élément) 
fruits.push("Kiwi");

Ajoute un ou plusieurs éléments à la **fin** du tableau.

.pop() 
let dernier = fruits.pop();

Supprime et retourne le dernier élément du tableau.

.unshift(élément) 
fruits.unshift("Fraise");

Ajoute un élément au **début** du tableau (re-indexe tous les éléments).

.splice(index, nbSuppr, ...items) 
fruits.splice(1, 1, "Melon");

Méthode tout-en-un. Permet d'**ajouter**, **supprimer** ou **remplacer** des éléments à n'importe quel index. L'exemple supprime 1 élément à l'index 1, et insère "Melon".

Méthodes d'Itération & Immuables
.map(), .filter(), .forEach()
.forEach(callback) 
fruits.forEach(item => {
  console.log(item);
});

Exécute une fonction **une fois pour chaque élément**. (N'a pas de valeur de retour).

.map(callback) 
let majuscules = fruits.map(item => item.toUpperCase());

Crée un **nouveau tableau** en appliquant une fonction à chaque élément du tableau original. (Immuable).

.filter(callback) 
let longs = fruits.filter(item => item.length > 5);

Crée un **nouveau tableau** contenant uniquement les éléments qui répondent à la condition testée dans la fonction.
