PHP

affichage

  • difference entre echo et print:
    • echo peut prendre plusieurs paramètres (bien que cela soit rare)
    • print ne peut prendre qu'un seul argument et retourne toujours 1
    • echo est légèrement plus rapide que print
  • echo
    • <?php
    echo "Bonjour le monde!";
    echo "\n"; pour sauter une ligne
    ?>
  • print
    • <?php
    print "Bonjour le monde!";
    print "\n"; pour sauter une ligne
    ?>

Chaînes de caractères

  • strlen() Longueur d’une chaîne.
  • strtoupper() / strtolower() Met en majuscules / minuscules.
  • ucfirst() / ucwords() Met la première lettre en majuscule.
  • str_replace() Remplace un mot par un autre.
  • substr() Extrait une sous-chaîne.
  • strpos() Cherche la position d’un mot dans une chaîne.
  • trim() Supprime les espaces en début/fin.
  • explode() Découpe une chaîne avec un séparateur.
  • implode() Joint un tableau avec un séparateur.
  • exemples: 
    <?php
    $texte = " Bonjour le monde! ";
    echo strlen($texte);  // affiche 18
    echo strtoupper($texte);  // affiche " BONJOUR LE MONDE! "
    echo trim($texte);  // affiche "Bonjour le monde!"
    ?>

variables

  • Commencent toujours par $
  • Peuvent contenir des lettres, chiffres et _
  • Ne peuvent pas commencer par un chiffre
  • Sensibles à la casse ($Nom et $nom sont différents)
  • exemples: 
    <?php
$nom = "Alice";
$age = 30;
echo "Nom: " . $nom . ", Âge: " . $age;
?>

elements

  • array (tableau)
    • count() Retourne le nombre d’éléments.
    • in_array() Vérifie si une valeur existe dans un tableau.
    • array_push() Ajoute un élément à la fin.
    • array_pop() Supprime le dernier élément.
    • array_shift() / array_unshift() Supprime / ajoute au début.
    • array_merge() Fusionne plusieurs tableaux.
    • array_slice() Extrait une partie du tableau.
    • array_keys() / array_values() Retourne les clés / valeurs.
    • foreach ($array as $val) Parcourt un tableau.
    • implode() Transforme un tableau en chaîne.
    • explode() Transforme une chaîne en tableau.
    • sort() / rsort() Trie un tableau (croissant / décroissant).
    • array_filter() Garde les éléments selon une condition.
    • array_map() Applique une fonction à chaque élément.
    • array_reduce() Réduit un tableau à une seule valeur.
    • array_search() Cherche une valeur et retourne sa clé.
    • array_reverse() Inverse l’ordre des éléments.
    • exemples: 
      <?php
    $nombres = [1, 2, 3, 4, 5];
    array_push($nombres, 6);  // ajoute 6
    print_r($nombres);  // affiche [1, 2, 3, 4, 5, 6]
    ?>
      •
  • liste
    • indexée numériquement
    • commence à 0
    • accès par position: $list[index]
    • modifie par position: $list[index] = valeur
    • ajoute un élément: $list[] = valeur
    • longueur: count($list)
    • parcourir: foreach ($list as $val) { ... }
    • vérifier existence: in_array(valeur, $list)
    • supprimer le dernier élément: array_pop($list)
    • supprimer le premier élément: array_shift($list)
    • trier: sort($list)
    • trier décroissant: rsort($list)
    • fusionner: $newList = array_merge($list1, $list2)
    • extraire une partie: $subList = array_slice($list, start, length)
    • obtenir les clés: $keys = array_keys($list)
    • obtenir les valeurs: $values = array_values($list)
    • transformer en chaîne: $str = implode(", ", $list)
    • créer à partir d'une chaîne: $list = explode(", ", $str)
    • appliquer une fonction: $newList = array_map('fonction', $list)
    • filtrer selon une condition: $filtered = array_filter($list, 'fonction')
    • réduire à une valeur: $result = array_reduce($list, 'fonction')
    • chercher une valeur: $key = array_search(valeur, $list)
    • inverser l'ordre: $reversed = array_reverse($list)
    • parcourir avec clé: foreach ($list as $index => $val) { ... }
    • vider la liste: $list = []
    • vérifier si vide: empty($list)
    • fusionner des listes: $merged = array_merge($list1, $list2)
    • diviser une liste: list($a, $b) = $list;
    • compter les occurrences: $counts = array_count_values($list)
    • obtenir une clé aléatoire: $key = array_rand($list)
    • mélanger la liste: shuffle($list)
    • convertir en tableau associatif: $assoc = array_combine($keys, $list)
    • convertir en liste indexée: $indexed = array_values($assoc)
    • supprimer des doublons: $unique = array_unique($list)
    • trier selon une fonction: usort($list, 'fonction')
    • trier en conservant les clés: asort($assoc)
    • trier en ordre inverse en conservant les clés: arsort($assoc)
    • fusionner des tableaux multidimensionnels: array_merge_recursive($array1, $array2)
    • obtenir la première clé: $firstKey = array_key_first($assoc)
    • obtenir la dernière clé: $lastKey = array_key_last($assoc)
    • exemple: 
      <?php
$fruits = array("pomme", "banane", "orange");
echo $fruits[0];  // affiche "pomme"
                        ?>
  • objet
    • <?php
class Personne {
    public $nom;
    public $age;
    function __construct($nom, $age) {
        $this->nom = $nom;
        $this->age = $age;
    }
}
$personne = new Personne("Charlie", 25);
echo $personne->nom;  // affiche "Charlie"
                        ?>

fonctions

  • Déclare une fonction: function nom($param1,$param2,...){code}
  • anonymous function: $f = function() {};
  • Peut prendre des paramètres
  • Peut retourner une valeur avec return
  • global Utilise une variable globale dans une fonction.
  • static Conserve la valeur d’une variable locale entre appels.
  • call_user_func() Appelle une fonction dynamiquement.
  • exemple: 
    <?php
function additionner($a, $b) {
    return $a + $b;
}
echo additionner(5, 3);  // affiche 8
?>
